# ShopVerse
ShopVerse is a simple e-commerce frontend web app that displays products and basic navigation between pages.

# purpose
I built this project to practice React, component-based UI, routing, and overall project structure.

# How to run
npm install  
npm run dev  

# Tech used
- React  
- TypeScript  
- Vite  
- Tailwind CSS  
- React Router  
- TanStack Query
